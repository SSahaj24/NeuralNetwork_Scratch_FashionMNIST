# NeuralNetwork_Scratch_FashionMNIST

This repository contains a NumPy implementation of a neural network for the Fashion MNIST classification task 

I named it TensorSlow (haha) as a reference to TensorFlow - the library structure and API for classes/functions have been taken or heavily inspired from TensorFlow

I've tried my best to keep the code modular, following the separation into files similar to the actual TensorFlow library, but with a limited feature set
